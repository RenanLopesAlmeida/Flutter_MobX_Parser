# About

This project focus on how to create a simple login/SignUp feature using Flutter, mobx, Get_It and Parse Server as backend with some validations

## What did I Use

- Flutter
- MobX
- Parse Server
- Get It
- Phone Formatter
- Regex to validate Email field
